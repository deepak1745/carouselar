# Carouselar

## Features

- Responsive
- Lazy Loading

## Installation

``bower install carouselar``

## Basic Usage

```
...
  $scope.myImages = [
    'img1.png',
    'img2.png',
    'img3.png',
    'img4.png',
  ];
...
```

```
<carouselar max-image="3" images="myImages" />
```
